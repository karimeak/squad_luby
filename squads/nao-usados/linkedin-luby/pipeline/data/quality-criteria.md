# Quality Criteria — LinkedIn Luby Squad

## Scoring System

- Cada critério: 1-10
- Hard rejection trigger: qualquer critério abaixo de 4/10
- APPROVE: média ≥ 7,0 E nenhum critério abaixo de 4
- CONDITIONAL APPROVE: média ≥ 7,0 mas critério não-crítico entre 4-6
- REJECT: média < 7,0 OU qualquer critério abaixo de 4

---

## Critérios — LinkedIn Post

| Critério | Descrição | Peso |
|---|---|---|
| Hook (scroll-stop) | As 2 primeiras linhas (~210 chars) param o scroll e criam urgência ou curiosidade? | Crítico |
| Voz pessoal | O post é escrito em primeira pessoa com perspectiva genuína, não genérica? | Alto |
| Estrutura | Hook → história → insights → CTA segue a estrutura definida? | Alto |
| Especificidade | Dados, números e exemplos concretos em vez de afirmações vagas? | Alto |
| CTA genuíno | A pergunta final é específica e convida resposta real (não retórica)? | Médio |
| Relevância Luby | O conteúdo conecta com o negócio e expertise da Luby naturalmente? | Alto |
| Sem links no corpo | Zero links no corpo do post? | Crítico |
| Hashtags | 3-5 hashtags na última linha, mix de amplas e nicho? | Médio |
| Anti-corporativo | Zero jargão corporativo ("synergy", "leverage", "at the end of the day")? | Alto |
| Tamanho | Total < 3.000 chars; parágrafos de 1-2 frases? | Médio |

## Critérios — LinkedIn Carousel

| Critério | Descrição | Peso |
|---|---|---|
| Hook slide | Slide 1 cria curiosidade suficiente para o usuário clicar para ver mais? | Crítico |
| Densidade por slide | Máximo 25-30 palavras por slide; 1 conceito único por slide? | Alto |
| Progressão lógica | Os slides constroem um argumento ou ensinam passo a passo? | Alto |
| CTA no slide final | Último slide tem chamada para ação clara e específica? | Médio |
| Consistência visual | Descrição de design é limpa, usa paleta Luby, fontes grandes? | Médio |
| Completude | 10-15 slides (não menos que 8, não mais que 18)? | Médio |
| Coerência com post | O carrossel expande o mesmo ângulo do post ou é complementar? | Alto |

## Critérios — LinkedIn Article

| Critério | Descrição | Peso |
|---|---|---|
| Headline SEO | Keyword no início, 60-100 chars, gera curiosidade específica? | Crítico |
| Introdução hook | Primeiras 2-3 frases forçam o clique para ler mais? | Crítico |
| Estrutura H2/H3 | 3-5 seções com subheadings informativos (não criativos)? | Alto |
| Evidência por seção | Cada seção tem pelo menos 1 dado, caso ou exemplo concreto? | Alto |
| Takeaway acionável | Cada seção termina com algo que o leitor pode implementar hoje? | Alto |
| Word count | 1.500-2.000 palavras (não abaixo de 1.200)? | Médio |
| Parágrafo curto | Parágrafos de 2-4 frases máximo; sem walls of text? | Médio |
| CTA único | Um único CTA no final (não múltiplos CTAs espalhados)? | Médio |

## Critérios — Qualidade Geral (todos os formatos)

| Critério | Descrição |
|---|---|
| Anti-commodity | O conteúdo poderia ter sido escrito por qualquer outra empresa? Se sim: REJECT |
| Voz Luby | Tom técnico-humano, confiante mas acessível — não corporativo nem casual demais |
| Precisão factual | Dados e afirmações são verificáveis e corretos? |
| Consistência de ângulo | Todo o conteúdo (post + carrossel + artigo) usa o mesmo ângulo selecionado? |

---

## Hard Rejection Triggers (qualquer um = REJECT imediato)

1. Hook não passa no scroll-stop test (score < 4/10)
2. Links no corpo do post
3. Jargão corporativo pesado ("synergy", "leverage", "scalable solutions", "paradigm shift")
4. Conteúdo poderia ser usado por qualquer concorrente sem modificação
5. Article abaixo de 1.200 palavras
6. Carousel com menos de 8 slides
7. Dados falsos ou não verificáveis usados como prova
